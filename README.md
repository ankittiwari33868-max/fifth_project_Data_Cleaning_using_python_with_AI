# fifth_project_Data_Cleaning_using_python_with_AI
# 🤖 AI-Powered Data Cleaning Project

## 👋 About the Project
This project focuses on automating the data cleaning process using AI techniques and Python.  
Raw datasets often contain missing values, duplicates, inconsistent formats, and errors.  
This project builds an intelligent pipeline to clean and prepare data for analysis.

---

## 🎯 Objective
To develop an automated data cleaning system that:
- Reduces manual effort  
- Improves data quality  
- Prepares datasets for analysis and visualization  

---

## 🛠️ Tools & Technologies Used
- Python 🐍  
- Pandas  
- NumPy  
- AI-based logic / automation  
- Jupyter Notebook  

---

## ⚙️ Features

✔ Remove duplicate rows  
✔ Handle missing values  
✔ Clean text data (trim spaces, fix casing)  
✔ Standardize column names  
✔ Detect and fix inconsistent data  
✔ Categorize data (High / Medium / Low values)
